# DeepCSAT – Ecommerce Customer Satisfaction Prediction

## Overview
DeepCSAT is a Machine Learning project that predicts Customer Satisfaction Score (CSAT) for ecommerce customer support interactions. The project analyzes historical support data and builds predictive models to estimate customer satisfaction levels.

## Objective
The goal of this project is to analyze customer support interaction data and build a predictive model that can estimate CSAT scores based on factors such as support channel, response time, handling time, and product category.

## Features
- Exploratory Data Analysis (EDA)
- Data preprocessing and cleaning
- Feature engineering
- Machine Learning model training
- CSAT prediction using classification algorithms
- Streamlit web application for prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
